# Whisper

---

## 🚀 Features

- 🎙️ Hide Text and Image files (PNG)
- Text -> Text
- Text -> Image
- Text -> Audio
- Image -> Image
- 🧠 Extract hidden content
- from Text -> Text
- from Image -> Text
- from Audio -> Text
- from Image -> Image
- 🖥️ User-friendly interface
- 📝 Save data and set password
